If no title can be determined from a story, our documentation gives a list of possible choices.
First line of text/dialogue in the first panel of the story, ignoring reoccurring blurbs or promotional text.
Only features for the series language can be selected. Enter "<space>[<text>" to search the disambiguation. [l] marks letter page features, [i] stands for in-house columns, [a] stands for ad features.
Only select a feature logo if it is present directly on the sequence. The feature corresponding to the selected feature logos will be added automatically.
Only story arcs for the series language can be selected. Enter "<space>[<text>" to search the disambiguation.

 Creator Credits:
For sequence types with non-optional fields the corresponding no-field is to be checked in case the type of credit does not apply.
If a credit field is not required for a sequence type it can be left unset or blank.
Check this box if script or plot does not apply to this sequence, e.g. for a cover or single-page illustration, and leave the script field blank.
Check this box if pencils does not apply to this sequence, e.g. for an unillustrated text article, and leave the pencils field blank.
Check this box if inks does not apply to this sequence, e.g. for a story colored straight from pencils, and leave the inks field blank.
Check this box if colors does not apply to this sequence, e.g. for a black-and-white comic, and leave the colors field blank.
Check this box if letters does not apply to this sequence and leave the letters field blank.
Check this box if there is no separate editor for this sequence. This is common when there is an editor for the whole issue.

 Characters:
Characters can be entered in several ways:
  • select several characters in the first autocomplete Appearing characters, each without additional details about the appearance,
  • select a group in the second autocomplete Group with the appearing group members in the third autocomplete Group members, again without additional details about the appearance,
  • each character (with its groups) separately, where additional details about the appearance can be entered.
  • the old text field for characters,
For a selected superhero the linked civilian identity will be added automatically. In the later character display both are shown together as Superhero [Civilian Name], but they appear as two entries in this editing form. You can also select an alternative civilian name, if needed.
Note that data from a) and b) will appear in section c) after a save.

For newspaper strips the format is from <comic strip> (<syndicate name>) <date>, example:

The notes field is for information that does not fit in other fields, but is worth noting. For visual structure, one can use the markdown syntax. E.g. **bold** or *italic* for bold and italic text, respectively. Lists start with a *, or are numbered, and can be nested by indenting with spaces. Links have the form [Link](http://a.com). Internal GCD links to other objects in the database have the general format [gcd_link_object_name](id), e.g. [gcd_link_issue](442). The generated links show under the display name of the object, or one uses the form [gcd_link_name_object_name](id){Link Title}.
